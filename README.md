# 🎙 Грамофонче — каталог по дължина

Подреден каталог на записите от [Грамофончето](https://gramofonche.chitanka.info/)
(копие на сбирката на Свилен Добрев в chitanka.info) — приказки, песнички и радиотеатър
от БалканТон плочи и касети.

**Целта:** да виждаш **дължината на всяка приказка предварително** и да филтрираш
по време (до 10 мин / 10–30 / 30–60 / над час), автор, качество и заглавие —
преди да пуснеш нещо на децата.

Раздел „Приказки" в оригинала: ~1119 записа / 13 GB / 310 часа.

## Употреба

```bash
pip install requests beautifulsoup4

python3 gramofonche.py scrape          # обхожда /prikazki/ -> data/albums.json (~15-20 мин при delay 1s)
python3 gramofonche.py scrape --all    # + песнички + за възрастни
python3 gramofonche.py build           # -> index.html (single-file, offline, mobile-first)
```

`index.html` работи офлайн — данните са вградени. Пусни го през GitHub Pages
и каталогът е достъпен от телефона.

Автоматично обновяване: `.github/workflows/update.yml` (седмично).

## Локално сваляне (лична употреба)

```bash
# кратките приказки до 20 минути, в папки "015мин - Заглавие":
python3 gramofonche.py download --max-min 20 --out ~/Prikazki

# само Андерсен:
python3 gramofonche.py download --match андерсен --out ~/Prikazki
```

Папките се именуват `NNNмин - Заглавие`, така че дължината се вижда
директно във файловата система, сортирана по подразбиране.

## ⚠️ Важно

**Не качвай MP3 файловете в публично repo.** Записите са издания на БалканТон
и др. — препубликуването им е нарушение на права (и 13 GB не се побират в GitHub).
Това repo съдържа **само метаданни и линкове** към Грамофончето; свалянето с
`download` е за лична/семейна употреба на собственото ти устройство.

Скриптът пази паузa между заявките (`--delay`) — не натоварвай chitanka.
